## Описание программы


Book Tracker - это GUI-приложение для управления списком прочитанных книг. Приложение позволяет добавлять книги, фильтровать их по жанру и количеству страниц, сохранять и загружать данные в формате JSON, а также удалять отдельные записи


### Возможности:
✅ Добавление книг с проверкой всех полей
✅ Фильтрация книг по жанру
✅ Фильтрация книг по количеству страниц (более 200)
✅ Сохранение данных в JSON-файл
✅ Загрузка данных из JSON-файла
✅ Удаление выбранной книги из списка
✅ Валидация вводимых данных (проверка пустых полей, корректность количества страниц)

## Инструкция по запуску
- python main.py

### Требования:
- Python 3.7 или выше
- Установленная библиотека Tkinter (обычно идёт в комплекте с Python)
- Установленная библиотека json


### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
   ```bash
   cd myProject
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```


## Примеры использования


### Добавление книги:
Заполните все поля формы:
Название книги
Автор
Жанр
Количество страниц
Нажмите кнопку "Добавить книгу"
Книга появится в таблице в нижней части окна

### Сохранение и загрузка:
- Нажмите **"Сохранить в JSON"** - данные сохранятся в файл `books.json`




## Валидация данных


Приложение проверяет:
- Все поля должны быть заполнены
